项目只能放到www根目录，不要建立文件夹

1. 移动端实现点击链接自动拨打电话的方法

只需要在想显示的地方添加如下代码即可：
<a href="tel:4006666666">联系电话</a>
在页面中看到“联系电话”的字样，点一下，就会自动跳到拨打电话的页面

2. Mail
邮件就和普通的html一样使用mailto

3. 短信
如果是需要调用短信的接口，可以将链接写成下面的格式：sms:<phone_number>[,<phone-number>]*[?body=<message_body>]
